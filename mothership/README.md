# 情报母舰服务

母舰是组织管理员使用的独立服务。它接收多个“情报哨站”按用户授权上传的结构化情报，提供协作空间、邀请密钥、成员授权、节点管理、重要预警、人工复核、处置状态、原文申请和审计日志。

默认不会接收发送者身份或完整聊天原文。

本地复赛演示可以直接运行：

```powershell
python -m mothership.bootstrap
```

启动器会生成一次性显示的管理员密钥。将母舰地址和该密钥填入客户端“设置 → 情报母舰 → 管理员工作台接入”，再到母舰创建协作空间和学生邀请。

生产部署使用环境变量：

```powershell
$env:MOTHERSHIP_ADMIN_TOKEN = "请替换为至少32位随机令牌"
$env:MOTHERSHIP_RETENTION_DAYS = "90"
$env:MOTHERSHIP_JOIN_BASE_URL = "https://你的学生端地址/settings"
python -m uvicorn mothership.app:app --host 0.0.0.0 --port 8010
```

`MOTHERSHIP_JOIN_BASE_URL` 决定邀请链接和二维码打开的学生端入口；本机演示默认指向 `http://127.0.0.1:8000/settings`。生产部署必须使用可访问的 HTTPS 地址、持久化磁盘和高强度管理员密钥。管理员创建协作空间后生成一次性显示的邀请密钥；学生接受邀请并确认范围后获得仅保存在本机的成员密钥。邀请密钥可轮换，成员可暂停或撤销，管理员也可停用单个哨站。

完整原文不在常规同步模型中。管理员只能对单条情报说明理由并发起申请，学生逐条批准后才会上传；批准内容默认 7 天后清除。
